Neirochef — это современное веб-приложение, созданное для удобного управления рецептами, отслеживания запасов продуктов, составления меню на неделю и минимизации пищевых отходов. Проект разработан с использованием Python и Django, чтобы обеспечить сочетание удобства и мощных возможностей бэкенда.

---

## Основные функции

- 🥗 **Управление рецептами**:
  - Добавление, редактирование и организация рецептов с шагами приготовления и списками ингредиентов.
  - Классификация рецептов по типам блюд (завтрак, обед, ужин).

- 🛒 **Отслеживание запасов**:
  - Учет продуктов с указанием сроков годности.
  - Уведомления о приближении истечения срока годности.

- 📅 **Планирование меню**:
  - Планирование питания на неделю.
  - Автоматическое создание списков покупок.
